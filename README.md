# DungeonSecrets

Dungeon secrets is 2D Swing based game with external database. The game is inspired by the popular game Dungeons and Dragons.
The project was developed as part of the group activity "Summer of Java 21".


## Used technologies
- Java SE 
- Swing
- Maven
- REST
- MySQL
- GitHub
